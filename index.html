Sicherheits-Checkliste

1. Sicherheitslücken schließen
1.1 Existiert ein definierter Prozess für Softwareaktualisierungen?
1.2 Wird die Sicherheit regelmäßig geprüft?
2. Benutzerzugänge absichern
2.1 Sind sichere Passwörter für alle Benutzer erforderlich?
2.2 Wird eine Multi-Faktor-Authentifizierung genutzt?
3. Datensicherung durchführen
3.1 Werden regelmäßige Backups durchgeführt?
4. Gefahrenbewusstsein schaffen
4.1 Gibt es Schulungen zum Thema IT-Sicherheit für Mitarbeiter?
5. Netzübergänge absichern
5.1 Wird eine Firewall genutzt?
6. Schadprogramme abwehren
6.1 Wird eine Anti-Virus-Software verwendet?
7. Notfallplan erstellen
7.1 Gibt es einen dokumentierten Notfallplan?
8. Inventarisieren und dokumentieren
8.1 Gibt es eine vollständige IT-Inventarisierung?
8.2 Werden alle Änderungen dokumentiert?
